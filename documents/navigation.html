Navigation

Creating a responsive navigation bar is very simple in WorkAround. Start with a <nav> tag and give it a class="navbar". Inside the nav create an un-ordered list with the desired navigational items. Each <li> gets a class="navbar-item". Wrap the content of the <li> elements in link tags.

<nav class="navbar">
<ul>
<li class="navbar-item"><a href="#">Page 3</a></li>
<li class="navbar-item"><a href="#">Page 2</a></li>
<li class="navbar-item active"><a href="#">Page 1</a></li>
</ul>
</nav>

Add a class="active" to an <li> item to emphasize the active page.

Note that the default color of the navigation bar is $primary-color, which lightens 4-fold upon hover. The text is the $default-color. The navigation bar is by default full-width. Also, note that items float right by default at screen sizes greater than 480px. At smaller screen sizes, the navigation bar items stack on top of each other. (Try it out!) Visit the Breakpoints page for more information. To modify any of these styles, please see the Sass file _components.scss, starting on line 57.
