This is a footer.

To give a webpage a footer, simply use the <footer> tag at the end of the body of the page and place whatever content you'd like to include inside the tag. By default, content is center-aligned with a 2em padding all around and top and bottom margins. To modify any of these styles, please see the Sass file _components.scss starting on line 50.
