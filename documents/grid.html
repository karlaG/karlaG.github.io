Grid

The grid system in WorkAround is a 6-column grid system, allowing anywhere from 2 to 6 columns at a time. To create a grid, start with a row and inside this create columns as shown below. Content is placed within columns and only columns may be immediate children of rows.

<div class="row">
<div class="col-3">
<h3>Column</h3>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vestibulum condimentum lacinia molestie. Donec consequat arcu est.</p>
</div>

</div>
</div>

For 2 columns, add the class "col-2" to each column. For 3 columns, "col-3" and so on. The columns in the grid are equal width and have equal spacing between each other to fit evenly within the row. A row spans the full width of its parent container.

There are two breakpoints asscoiated with the grid system. The first is that for screen widths below 970px, a 6-column grid will wrap onto 2 lines of 3 columns each. The second is that for screen widths below 640px, all size grids will stack their columns in consecutive order into a single column. (Make your browser smaller and try it out!) See the Breakpoints page for more information. To modify the grid system, please see the Sass file _components.scss starting on line 22 and _variables.scss starting on line 15.
