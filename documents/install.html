Installation Instructions

First of all, thank you for using WorkAround. I promise you won't be disappointed! Let's begin by downloading WorkAround.




The download contains a zipped file with CSS and Sass folders. To use WorkAround in your HTML code, include the following in the <head> of your document, modifying the path as needed:

<link rel="stylesheet" href="css/workaround.css">

You can always override any WorkAround CSS by writing your own and giving it preference. However, you can also recompile the Sass files with any edits you may make. Partial Sass files include definitions of variables, component styles and responsive design. WorkAround is a mobile-first application.


And that's it! You're ready to start using WorkAround.
