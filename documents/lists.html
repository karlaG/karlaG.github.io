Listgroup

To define a listgroup, start with a <div class="listgroup"> wrapped around an un-ordered list. Add <li> elements as desired to define the content.

<div class="listgroup">
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>
</div>
  • Item 1
  • Item 2
  • Item 3

Add the class="label" to a listgroup <li> element to create a section header. <li> elements can also be links. To learn more about links visit the Text page.

<div class="listgroup">
<ul>
<li class="label">Header</li>
<li>Item</li>
<li> <a href="#">Link</a> </li>
</ul>
</div>
  • Header
  • Item
  • Link

To modify any of these styles, please see the Sass file _components.scss, starting on line 106.
