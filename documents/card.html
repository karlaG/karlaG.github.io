Card

A card is a block level element that has a top and bottom margin of 1em, as well as 1em paddng all around. It's default background color is a 50% transparent $secondary-color. Add the card class to any element to achieve these effects. To modify the card class, please see the Sass file _components.scss starting on line 44.

<p class="card">This is a card.</p>
