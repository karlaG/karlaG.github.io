Buttons

Buttons can be declared two ways:

<a href="#" class="button-link">
<button>Button Link</button>
</a>

<button>Regular Button</button>

Button links may be used to link to different pages or anchors within pages. Regular buttons can be used for forms or other functions (may require adding javascript).


They come in two styles:

<button class="button-primary">Button Primary</button>
<button class="button-secondary">Button Secondary</button>

To modify any of these styles, please see the Sass file _components.scss, starting on line 121. The primary button style uses the $secondary-color and lightens this color 4-fold upon hover, while the secondary button style starts with a transprent background, filling in with $tertiary-color upon hover and changing the text to $default-color.
