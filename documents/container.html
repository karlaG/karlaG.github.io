Container

A container is a simple way to begin to lay out objects on your page. Giving a block-level element the class container, automatically centers that object on the page and it gives it a margin of 10% of the screen width on each side. To modify the container class, please see the Sass file _components.scss starting on line 18.

//50 x 50 px orange box
<div class="container" style="background-color:orange;height:50px;width:50px;"></div>
