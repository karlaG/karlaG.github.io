Text

By default, all text in WorkAround is a member of the font-family Hind Guntur, the value of $font, a color $font-color, and a font size $font-size. Find these declarations in the Sass file _components.scss starting on line 11 and the definitions in _variables.scss starting on line 4.


WorkAround also comes with default styling for <h1>,<h2>, <h3>, <p>, and <a> tags. Try making the screen smaller. The typography should automatically adjust to fit the screen size. To learn more about this visit the Breakpoints page.
